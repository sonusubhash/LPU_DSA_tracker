# Search a 2D Matrix
- https://leetcode.com/problems/search-a-2d-matrix/description/
``` python 
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        if not digits:
            return [1]

        if digits[-1] < 9:
            digits[-1] += 1
            return digits
        else:
            return self.plusOne(digits[:-1]) + [0]
```
